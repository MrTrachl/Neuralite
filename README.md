## 🤔 Why Neuralite?

Courses offered online are often overly priced for suspiciously low value quality in material and design — seemingly made with everything in mind except the user (and their wallet). Neuralite aims to flip this right on its head by not only providing quality content through the power of AI, but pay **you** to learn with a purely exam and learning focused platform tailored to none other than... **you**.

## 🎨 Design Decisions

**Deep Ocean Void UI** &mdash; Neuralite utilizes a extremely dark blue pallete combined with a vibrant purple theme, optimal for a platform where users would spend long durations of time engaging with content. 

**Simple and Effective Structure** &mdash; Most of the core pages are relatively simple, focusing on keeping just what is needed, while AI provides the learning experience and generating template based exams based on your learning curve and objective; thus as expected, less html bulk was needed and thus, simpler pages.

**Exam HUD Toolbar** &mdash; The exam page has one of the most interesting and potentially controversial components as well &mdash; a glass HUD toolbar that sits at the bottom of the page and consequentally, over the tail end of the questions. This was done was done with intention. If a whole bar div was placed at the bottom, it would naturally block out the scrolling content beneath it; instead I opted for the unique glass toolbar effect which I found to be visually interesting and unique.

## 🚀 Run Locally

Since Neuralite is a static front-end project (no backend), getting it running is quick.

**Option 1 — Just open it**

Clone the repo and open the entry file directly in your browser:

```bash
git clone https://github.com/MrTrachl/Neuralite.git
cd Neuralite
```

Then double-click `index.html` (or drag it into your browser).
